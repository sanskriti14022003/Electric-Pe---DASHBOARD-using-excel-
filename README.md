# Electric-Pe---DASHBOARD-using-excel-
is a detailed performance analysis of the lead-to-delivery funnel for a product. It evaluates 1,542 Total Leads using KPIs like a 24.6% Conversion Rate (Lead $\rightarrow$ Delivery), a 13.4-day Average TAT, and a 6.8% Cancellation Rate. The analysis is completed by a customer satisfaction score with an NPS of 31.
